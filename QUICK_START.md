# 🚀 QUICK START GUIDE

## Deploy Your Portfolio in 3 Steps!

### ✅ Step 1: Create GitHub Repository

1. Go to https://github.com/new
2. Name it: `Richmond-Antor.github.io` (EXACTLY this name)
3. Make it Public
4. Click "Create repository"

### ✅ Step 2: Upload Files

**Easiest Method:**
1. Click "uploading an existing file" on GitHub
2. Drag ALL files from this folder
3. Commit with message: "Initial portfolio"

### ✅ Step 3: Enable GitHub Pages

1. Go to repository Settings
2. Click "Pages" in sidebar
3. Under "Branch", select "main" and "/ (root)"
4. Click Save

### 🎉 Done!

Your site will be live at: `https://Richmond-Antor.github.io`

(Wait 5-10 minutes for first deployment)
